# ClubSphere - Backend

## Project Purpose

ClubSphere Backend provides REST APIs for managing clubs, users, events, memberships, and payments.

## Live Server

https://my-club-sphere-server.vercel.app

## Key Features

- RESTful API
- MongoDB database
- Firebase token verification
- Role-based authorization
- Stripe payment handling
- Admin moderation
- Search, filter & sorting APIs

## Technologies & Packages

- Node.js
- Express
- MongoDB
- Firebase Admin
- Stripe
- dotenv
- cors

## Setup

1. npm install
2. Create .env with DB & Stripe keys
3. npm run dev
